# Bitrix-Sums

A userscript that automatically calculates and displays hour and story point totals in Bitrix boards and sprints.

## Features

- Sums hours and story points in Bitrix boards
- Sprint tracking with remaining days support
- Task link copying (Markdown and Excel formats)
- Visual enhancements for Bitrix kanban boards

## Installation

1. Install a userscript manager in your browser:
   - [Tampermonkey](https://www.tampermonkey.net/) (Chrome, Firefox, Safari, Edge)
   - [Greasemonkey](https://www.greasespot.net/) (Firefox)

2. Click to install: [bxSums.js](https://mobimedia.github.io/bxSums/bxSums.js)

3. The script will automatically run on Bitrix sites matching `https://bitrix.*.de/*`

## Usage

Once installed, the script runs automatically when you visit a Bitrix board or sprint page. It will:
- Calculate and display totals for hours and story points
- Add quick-copy buttons for task links
- Enhance the visual display of cards

## Requirements

- A modern web browser
- Tampermonkey or Greasemonkey extension
- Access to a Bitrix instance
